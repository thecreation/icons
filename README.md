### Installing

Install lerna globally.

```
npm i -g lerna
```

### Get Started

```
npm install
npm run bootstrap
npm run update
git submodule init
git submodule update
git submodule foreach git pull origin master
```

### Add new module

Refer to this [https://github.com/thecreation/standard-icons/tree/master/tools/module-generator](document)

### Publish

```
npm run import
npm run optimize
lerna publish
```
